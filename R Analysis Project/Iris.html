This dataset includes 150 different flowers described by the sizes and width of two of their attributes: the petals and the sepals. These values are given in centimetres.
We can notice that the flowers present in this dataset are part of three different species: Setosa, Versicolor and Virginica. They are equitably distributed in this dataset: 50 records per species.
It will be interesting to study which characteristic(s) discriminates each species and to what extent.
To do this we will start by analyzing each species in relation to the others, in order to determine if one of the characters is not in the average. If so, we will analyse this character more precisely.

is one of the parameter of the setosa specie far from the mean¶

summarySetosa <- c(mean(iris[iris$Species=="setosa", "Sepal.Length"]),mean(iris[iris$Species=="setosa", "Sepal.Width"]),mean(iris[iris$Species=="setosa", "Petal.Length"]),mean(iris[iris$Species=="setosa", "Petal.Width"]))
summaryGlobal <- c(mean(iris[iris$Species!="setosa", "Sepal.Length"]),mean(iris[iris$Species!="setosa", "Sepal.Width"]),mean(iris[iris$Species!="setosa", "Petal.Length"]),mean(iris[iris$Species!="setosa", "Petal.Width"]))
compariason1 <- rbind(summarySetosa,summaryGlobal)

barplot(compariason1, 
        beside=T,
        names.arg=c("Sepal.Length","Sepal.Width","Petal.Length","Petal.Width"),
        ylim=c(0,6), 
        col=c("darkblue","red"))
legend("topright", 
       legend = c("Setosa", "not Setosa"), 
       fill = c("darkblue", "red"))
